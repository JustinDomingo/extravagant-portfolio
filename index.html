miniPost/Pigeon

In it's simplest terms, "miniPost" is a minimal social media application in which you can express your thoughts through small posts. You can create your account through the user registration/login feature that uses Node, Express, Session, and MongoDB authentication. When successfully logged in, you can create textual posts which will also validated through Node and Express. Once successfully validated, the posts will be pushed and saved into a MongoDB database. You are then free to edit, and delete your posts if you aren't happy with them. The application uses the view engine, "EJS".


Quackery

Quackery is a minimal chatroom application in which you can exchange limitless messages with your friends. To begin chatting, you must choose a username (any username will work). You will then be redirected to the main chatroom. Behind the scenes, the application is using Socket.io to send real-time requests back and forth to the Node server and emitting them to all users that are currently connected. The application also uses sessions so each user will have unique usernames and data.


Heat Check

Heat Check is a simple weather application to see the current weather data of your desired city. When you submit the city form, the application sends a "GET" request through the "fetch" JavaScript method to a free weather API. Afterwards, the API responds with the proper weather data depending on the city the user typed into the field.


The Harry Potter Quiz

The Harry Potter Quiz starts off as an easy and simple JavaScript quiz. The program saves your score in session. Afterwards, you are offered the ability to post your score to a MongoDB leaderboard. The app also uses Express and Node authentication to valiate users before pushing them into a MongoDB database. There is also a feature in which you can post comments to other users.


Rightshot

Rightshot is a simple vanilla JavaScript game. There are three targets that are displayed in the container and you must shoot the correct target. After you shoot a target, the three targets will reposition in random places. You will be timed 60 seconds and the quicker you shoot the correct target, the higher your score will be. Note that your accuracy will also be tracked.


Heat Check (React Ver.)

This is the React version of the "Heat Check" weather application. The application uses the React "useState" hook whenever someone submits their inputted city. The inputted data is then used for the API's query parameters to make a "fetch" GET request.
